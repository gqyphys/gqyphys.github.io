#### 背景
> 有时会遇到元素的宽度需要结合百分比和一个固定值来确定，e.g.

```
<div class="example">
    <a>111</a>
</div>
```
需要`.example`元素宽度为父元素的80%再加上10px

#### 解决方案
使用`calc()`，e.g.
```
.example
{
    <!--表达式中有“+”和“-”时，其前后必须要有空格-->
    width: calc(80% + 10px);   
}
```
